# UVA Department of Surgery — Resident Travel Policy

A web-based reference guide for categorical residents, fellows, and clinical instructors in the **University of Virginia Department of Surgery**.

## About

This site provides an easy-to-navigate summary of the Categorical Resident Travel Policy (Version 2026-05), including funding guidelines, travel rules, approval workflows, international travel requirements, and key contacts.

## Contents

| Tab | Description |
|-----|-------------|
| **Overview** | Policy summary, who is covered, and funding quick-reference cards |
| **Funding** | Funding limits by resident category and important funding notes |
| **Travel Rules** | Booking, lodging, meals, transportation, and receipt requirements |
| **International Travel** | Registry requirements, export controls, digital security, and sanctions guidance |
| **Process** | Step-by-step travel approval workflow and key deadlines |
| **Contacts** | Program coordinators by division and funding contacts with Outlook links |
| **Travel Request Form** | Resident Travel Information Form for coordinator booking |
| **Policy Document** | Full policy PDF — viewable inline with download option |

## Usage

Open `index.html` in any modern web browser. No internet connection is required except for the embedded Microsoft Form on the Travel Request Form tab.

## Contact

For questions about this site, contact the Division of Surgical Science.
